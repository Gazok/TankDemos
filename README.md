Tank Camera Demo
================

A small demo showing off some new features:
Camera movement, rotation, zooming
Mouse support (click and drag entities in one function call!)

To install and run:
    git clone git@github.com:Gazok/CamDemo.git
    cd CamDemo
    git submodule init
    git submodule update
    mkdir build
    cd build
    cmake ..
    make
    ./test
